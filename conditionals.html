📚 12 ingliz zamoni

Formula + Qachon ishlatilishi + Misollar + Testlar

🔀 Shart gaplari

Type 0

If + Present Simple, Present Simple

If you heat ice, it melts.

Type 1

If + Present Simple, will + V1

If it rains, I will stay home.

Type 2

If + Past Simple, would + V1

If I were rich, I would travel.

Type 3

If + Past Perfect, would have + V3

If I had known, I would have come.

📝 Testlar

1. If you heat ice, it _____ (melt).

2. If it rains tomorrow, we _____ (stay) home.

3. If I _____ (be) you, I would go.

4. If she _____ (study), she would have passed.

5. If I had known, I _____ (come).
